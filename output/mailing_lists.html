Users¶

If you use Gora, please subscribe to the Gora user mailing list. The Gora user mailing list addresses are :

Mailing list Subscribe Unsubscribe List Archive
Users list Subscribe Unsubscribe Search List

In order to post to the list, it is necessary to first subscribe to it.

Developers¶

If you'd like to contribute to Gora, please subscribe to the Gora developer mailing list. The Gora developer mailing list addresses are :

Mailing list Subscribe Unsubscribe List Archive
Dev list Subscribe Unsubscribe Search List

In order to post to the list, it is necessary to first subscribe to it.

Developers and Committers¶

If you'd like to see changes made in Gora's version control system then please subscribe to the Gora commit's mailing list. The Gora commit's mailing list is :

Mailing list Subscribe Unsubscribe List Archive
Commits Subscribe Unsubscribe Search List

In order to post to the list, it is necessary to first subscribe to it.
